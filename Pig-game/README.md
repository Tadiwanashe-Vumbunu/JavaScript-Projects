# Pig Game

A simple two-player dice game built with JavaScript, HTML, and CSS.

## How to Play

- Two players take turns rolling a dice.
- Each dice roll is added to the player's current score (**CURRENT**) for that round.
- A player can continue rolling to accumulate more points.
- Clicking **HOLD** adds the current score to the player's **total score** and passes the turn to the next player.
- If a player rolls a **1**, they lose all points accumulated in the current round, and the turn immediately switches to the other player.
- The first player to reach **100 points** wins the game.
- Click **NEW GAME** to reset all scores and start a new match.

## Features

- Roll Dice functionality
- Hold Score functionality
- Winner detection
- New Game reset option
- Two-player gameplay
